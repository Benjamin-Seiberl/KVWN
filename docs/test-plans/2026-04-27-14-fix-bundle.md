- Push notification delivery on real devices (only the trigger/send side is tested here).
- Visual polish: spacing, font rendering, shadow tint correctness at pixel level.
- Offline / service-worker behaviour.
- Invite-player edge function (#2) — requires separate Supabase/Edge Function tooling.
- AdminRollen sheet (role management) — no changes in this bundle.

- Timezone boundary (midnight rollover during testing session).
